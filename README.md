# 3Doors Website

Simple GitHub Pages landing page for **3Doors** by RameNiku.

## How to use

1. Put the final game archive here:

   `downloads/3Doors.zip`

2. Upload all files in this folder to a GitHub repository.
3. In GitHub, open **Settings -> Pages**.
4. Set source to **Deploy from a branch**.
5. Choose branch **main** and folder **/root**.
6. Open the generated GitHub Pages link.

## Files

- `index.html` - website page
- `assets/normal.png` - screenshot/art
- `assets/bones_whispers.png` - screenshot/art
- `assets/coffee.png` - screenshot/art
- `downloads/3Doors.zip` - placeholder path for the final game download
